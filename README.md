# Usage
<!-- usage -->
```sh-session
$ npm install -g tez-cli
$ tez-cli COMMAND
running command...
$ tez-cli (-v|--version|version)
tez-cli/0.0.1 win32-x64 node-v12.11.0
$ tez-cli --help [COMMAND]
USAGE
  $ tez-cli COMMAND
...
```

```

  $ tez-cli help

ARGUMENTS

OPTIONS
  --all  see all commands in CLI
```
